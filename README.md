# 网页记事本 📝

一个基于 Web 的纯文本记事本，无需安装，打开浏览器即可使用。支持多标签页、本地文件读写、深色/护眼主题、大纲导航、查找替换等功能。

## ✨ 特性

- **多标签页** – 同时编辑多个文档，Ctrl+左右箭头快速切换，鼠标滚轮滑动浏览
- **本地文件读写** – 使用 File System Access API 直接打开/保存 .txt 文件
- **查找与替换** – 支持正则、大小写、全字匹配、倒序查找
- **大纲导航** – 自动识别 Markdown 标题和中文章节标题，显示各章节字数
- **主题切换** – 浅色、深色、护眼（豆沙绿）、复古四种主题
- **字体与行高自定义** – 多种等宽/非等宽字体可选
- **自动保存** – 对已关联文件的标签，停止输入 10 秒后自动写回磁盘
- **撤销/重做** – 每标签独立撤销栈，切换标签不丢失历史
- **拖拽打开** – 支持拖拽 .txt/.md 文件到编辑器区域直接打开
- **快捷键** – 丰富的键盘快捷键，提升编辑效率

## 🚀 使用方法

1. 直接打开 `index.html` 即可使用（推荐使用 Chrome 或 Edge）
2. 或部署到任意静态服务器（GitHub Pages / Netlify 等）

### 新建文档
- 点击加号按钮或按 `Ctrl+N`，选择保存位置并输入文件名，创建新的 .txt 文件

### 打开文件
- 点击“文件 → 打开”或按 `Ctrl+O`，选择本地文本文件
- 也可以直接将文件拖入编辑器区域

### 保存
- `Ctrl+S` 保存当前文件
- `Ctrl+Shift+S` 另存为

### 查找替换
- `Ctrl+F` 查找，`Ctrl+H` 查找并替换

## 🔑 快捷键一览

| 快捷键                   | 功能               |
| ------------------------ | ------------------ |
| Ctrl+N                   | 新建文档           |
| Ctrl+O                   | 打开文件           |
| Ctrl+S                   | 保存               |
| Ctrl+Shift+S             | 另存为             |
| Ctrl+W                   | 关闭当前标签页     |
| Ctrl+Shift+W             | 切换自动换行       |
| Ctrl+F                   | 查找               |
| Ctrl+H                   | 查找和替换         |
| Ctrl+Z                   | 撤销               |
| Ctrl+Y                   | 重做               |
| Ctrl+= / Ctrl+- / Ctrl+0 | 字体放大/缩小/重置 |
| Ctrl+左右箭头            | 切换标签页         |

## ⚙️ 浏览器兼容性

- 推荐 Chrome 86+ 或 Edge 86+

- Safari 不支持 File System Access API
